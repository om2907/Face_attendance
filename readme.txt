-> dlib is required for installing face_recognition library


-> Steps to setup the project
	
	1)Install CMake
		-Download from CMake Official Website
		-Add CMake to your system PATH during installation.
	
	2)Install Visual Studio Build Tools
		-Download from Visual Studio Build Tools
		-Select "Desktop development with C++" and install.

	3)Install dlib
		pip install dlib    or    pip install dlib --no-cache-dir

	4)Install face_recognition
		pip install face_recognition

	5)Install all requirements
		pip install -r requirements.txt

	6)Run in localhost (in command prompt)
		streamlit run app.py
